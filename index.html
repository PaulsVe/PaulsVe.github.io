Spotify Listening Behavior Dashboard

This portfolio project showcases customer behavior analysis using my personal Spotify History Data with SQL and Power BI.

Project Summary

  • 📁 Data source: Spotify listening history
  • 🔄 Converted to: SQL database
  • 📊 Dashboard built in: Power BI
  • 🔍 Focus: Monthly listening patterns, top artists/tracks, session trends

SQL Schema & Queries

CREATE TABLE streaming_history (
  id INTEGER PRIMARY KEY,
  track_name TEXT,
  artist_name TEXT,
  end_time DATETIME,
  ms_played INTEGER
);

Sample Query:

SELECT artist_name, SUM(ms_played)/60000 AS total_minutes FROM streaming_history GROUP BY artist_name ORDER BY total_minutes DESC LIMIT 10;
